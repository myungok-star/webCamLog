# WebCamLog
https://fierce-sierra-85118.herokuapp.com/

##App Description
* WebCamLog let users to record video through a webcam on their device. User can log their diaries and notes in video format.

##Built With
* Node.js
* Express.js
* MongoDB
* AngularJS

##Dependency & Instructions
